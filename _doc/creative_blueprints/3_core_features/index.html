Built for the Solo Professional

What PayFlow Does

💼 Freelancer
🏠 Screen 1
Home Screen
Freelancer dashboard: "Create Payment Request" + "Quick UPI Payment" actions. Recent transactions list showing amount, service name, and status from local storage.
💳 Screen 2
Payment Screen
Amount input (required) + description/service name (optional). Tapping "Pay" converts amount to paise and opens Razorpay Checkout with optional contact/email prefill.
✅ Screen 3
Result Screen
Success (green): shows Payment ID + amount + description. Failed (red): shows failure status. Simple centered UI, no extras.
⚡
Razorpay UPI
Handles PAYMENT_SUCCESS and PAYMENT_ERROR events. Real checkout flow — no fake payments, no mock data.
💾
Local Transaction Log
Every payment saved: amount, description, paymentId, status, timestamp. No backend, no login, no cloud required in MVP.
🏗️
Clean Architecture
Structured for growth: screens/ · services/ · models/ · widgets/. Ready for invoice PDF, client management, cloud sync in future phases.
